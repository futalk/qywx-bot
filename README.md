# 企业微信报工提醒脚本

智能定时提醒系统，支持法定节假日和调休判断，自动向企业微信群发送工作日常提醒。

## 安装依赖

确保已安装 **Python 3.8+**，然后执行：

```bash
pip install -r requirements.txt
```

依赖说明：
- `requests` — 发送 HTTP 请求到企业微信 Webhook
- `chinesecalendar` — 自动识别中国法定节假日和调休

## 配置环境变量

Webhook Key 通过环境变量读取，不会硬编码在代码中。

### 方式一：直接设置环境变量

**Windows (CMD)**
```cmd
set QYWX_WEBHOOK_KEY=你的机器人密钥
```

**Windows (PowerShell)**
```powershell
$env:QYWX_WEBHOOK_KEY="你的机器人密钥"
```

**macOS / Linux**
```bash
export QYWX_WEBHOOK_KEY="你的机器人密钥"
```

### 方式二：使用 .env 文件（可选）

复制模板文件并填写密钥：

```bash
cp .env.example .env
```

然后编辑 `.env` 文件：
```env
QYWX_WEBHOOK_KEY=5218a69c-a715-457f-8fd8-f3a0b0cf7526
```

> **注意**：`.env` 文件已加入 `.gitignore`，不会被提交到 Git。

## 获取 Webhook Key

1. 在企业微信群中，点击右上角 **···** → **添加群机器人**
2. 选择「新建机器人」，填写名称
3. 复制 Webhook 地址中的 `key` 参数（例如：`key=xxxxxxxxxx`）

## 运行脚本

```bash
python qyex.py
```

启动后，脚本会：
1. 在控制台打印定时任务列表
2. 向企业微信群发送一条启动测试消息
3. 每 **30 秒** 检查一次时间，到达设定时间则发送对应提醒
4. 日志同时写入 `logs/reminder_年月.log`

### 停止服务

在运行窗口按 `Ctrl+C` 即可停止。

## 脚本特性

- 🕐 精准时间触发，防重复发送
- 📅 自动识别中国法定节假日和调休补班
- 📝 统一的任务配置表，易于增删改提醒规则
- ⚠️ 完善的错误处理和日志输出
- 🔄 消息发送失败自动重试（指数退避，最多3次）
- ✅ 启动时发送测试消息并显示当天状态
- 🔐 Webhook Key 通过环境变量读取，避免泄露

## 自定义提醒

打开 [`qyex.py`](qyex.py:68)，找到 `REMINDER_TASKS` 列表，按以下格式添加或修改任务：

```python
{
    "time": "11:45",           # 触发时间，格式 HH:MM
    "label": "🍽️ 【午饭时间】", # 消息标题
    "content": "内容正文",     # 支持字符串或返回字符串的函数
    "condition": is_workday,   # 触发条件函数，返回 True 才发送
}
```

## 注意事项

- 请妥善保管 Webhook Key，不要泄露给他人
- 确保电脑保持开机且脚本持续运行
- 建议使用后台进程或服务管理器（如 Windows 任务计划程序、Linux systemd）保持脚本长期运行
- 工作日定义：由 `chinesecalendar` 库自动判断，包含调休补班，排除法定节假日
